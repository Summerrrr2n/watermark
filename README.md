# Watermark pictures

批量给图片加水印，没有任何网络请求，保护证件、隐私、敏感图片安全。  

[Vue.js](https://cn.vuejs.org/index.html) + [element-ui](https://element.faas.ele.me/#/zh-CN/component/installation) + [JSZip](https://github.com/Stuk/jszip)

### 特性
- **批量**加水印
- 可配置大小、角度、间距、颜色
- 打包下载
- 大图预览
